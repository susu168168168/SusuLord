## 1. 游戏概述

### 1.1 游戏简介

**苏苏领主**是一款基于真实地理位置（LBS）的末日生存题材移动游戏。玩家扮演末日后的"开拓者"，通过真实行走圈定领地、探索废墟搜刮资源、建造设施重建家园，并与附近的其他开拓者进行交易与通讯。

### 1.2 核心玩法一句话

> **"用双脚丈量真实世界，在废土上建立属于自己的王国"**

### 1.3 游戏类型
- **主类型**: LBS（基于位置的服务）游戏
- **子类型**: 生存建造 / 探索收集 / 社交交易
- **参考游戏**: Pokemon GO、Ingress、饥荒

### 1.4 目标平台
- iOS (iPhone)
- 最低支持: iOS 15.0

### 1.5 艺术风格
**"希望朋克" (Hopepunk)**
- 废土之上的新生
- 破败建筑与绿色植物的对比
- 绝望中透出希望的色调
- UI风格：末日科技感 + 手绘温暖元素

## 2. 世界观与故事背景

### 2.1 时间线

| 时间 | 事件 |
|------|------|
| 2048年 | "终末战争"爆发，全球文明在72小时内崩塌 |
| 2048-2051年 | 黑暗时期，99%人口消失 |
| 2051年 | 幸存者走出废墟，《新开拓者协议》颁布 |
| 2051年至今 | 重建纪元，玩家故事开始 |

### 2.2 《新开拓者协议》

> **"凡以双足丈量之无主土地，皆为开拓者所有。"**

这是新世界的核心法则：
- 旧世界的土地所有权已经失效
- 用GPS记录的行走轨迹作为领地边界
- 闭合的轨迹所圈定的区域归开拓者所有
- 必须持续活跃，否则领地将被回收

### 2.3 世界设定

**环境特征：**
- 城市废墟与自然重生并存
- 旧世界基础设施残骸
- 卫星系统仍在运作（GPS可用）
- 通讯网络需要重建

**社会特征：**
- 无中央政府，开拓者自治
- 以物易物为主要交易方式
- 合作与竞争并存
- 领地即资产

### 2.4 玩家身份

玩家扮演一名刚从避难所醒来的**开拓者**：
- 没有过去，只有未来
- 需要从零开始建立家园
- 通过行动证明自己的价值
- 目标：成为最强大的领主

### 3.2 循环拆解

| 阶段 | 玩家行为 | 系统反馈 | 心理驱动 |
|------|----------|----------|----------|
| 外出行走 | 打开App开始移动 | GPS实时追踪显示轨迹 | 好奇心、运动需求 |
| 圈地占领 | 走一圈回到起点 | 领地确认+面积显示 | 所有权满足感 |
| 探索发现 | 经过真实地点 | 弹出POI发现提示 | 探索欲望 |
| 搜刮获取 | 点击搜刮按钮 | 资源入账+音效反馈 | 收集成就感 |
| 建造发展 | 消耗资源建造 | 建筑出现在领地 | 创造满足感 |
| 交易换物 | 发布/接受挂单 | 资源交换完成 | 社交需求 |
| 生存消耗 | 使用食物/水 | 状态恢复 | 紧迫感+策略性 |

### 3.3 时间分配设计

**典型玩家的一天：**

| 时段 | 活动 | 时长 |
|------|------|------|
| 通勤路上 | 探索搜刮POI | 20-30分钟 |
| 午休时间 | 管理背包、建造 | 5-10分钟 |
| 下班路上 | 圈地或探索 | 20-30分钟 |
| 晚间休闲 | 交易、社交、查看排行 | 10-20分钟 |

**总计日均游戏时长目标：1-1.5小时**

## 4. 游戏系统详细设计

### 4.1 GPS圈地系统

#### 4.1.1 圈地流程

```
1. 玩家点击"开始开拓"
2. 系统开始记录GPS轨迹
3. 玩家在现实中行走
4. 地图上绘制蓝色轨迹线
5. 玩家回到起点附近（30米内）
6. 系统自动闭合多边形
7. 验证合法性：
   - 无自交叉
   - 不与他人领地重叠
   - 速度合理（非作弊）
   - 面积达到最小要求
8. 验证通过 → 领地确认
9. 验证失败 → 提示原因
```

#### 4.1.2 圈地规则

| 规则 | 数值 | 说明 |
|------|------|------|
| 最小面积 | 1,000 m² | 约半个篮球场 |
| 最大面积 | 100,000 m² | 约10个足球场 |
| 闭合距离 | 30米 | 回到起点的容差 |
| 最大速度 | 15 km/h | 超过视为作弊 |
| 最小点数 | 4个 | 形成有效多边形 |
| 冷却时间 | 1小时 | 两次圈地间隔 |

#### 4.1.3 领地属性

| 属性 | 描述 |
|------|------|
| 名称 | 玩家自定义，默认"无名领地" |
| 面积 | 系统自动计算 |
| 边界 | 多边形顶点坐标 |
| 建筑槽位 | 根据面积决定可建造数量 |
| 活跃度 | 30天无活动则进入回收倒计时 |
| 交易开关 | 是否允许附近玩家交易 |
